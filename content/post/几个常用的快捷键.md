#### 打开Chrome跨域模式

要关闭浏览器所有页面才会执行

```kotlin
open -n "/Applications/Google Chrome.app" --args --disable-web-security  --user-data-dir=/Users/jeffrey/MyChromeDevUserData/
```

#### vscode快捷键

command + left/right: 一行左右切换

option + left/right: 单词切分，左右切换

option + delete: delete一个单词

shift + command + right（光标在行头时）: 选中整行

command + shift  + \ : 代码块前后跳转，光标要在括号后

#### 杀占用端口的进程

* mac平台：

lsof -i :port

kill -9 PID

* windows平台

```shell
netstat -ano|findstr "PID :3000" 
```

 3000是查询占用的端口，可以换成任意你想要查询的端口号

然后会显示出所有占用端口的进程，如下：

```shell
Proto Local Address Foreign Address State PID 
TCP 0.0.0.0:3000 0.0.0.0:0 LISTENING 18264
```

最后，用以下命令进行杀进程：

```shell
taskkill /pid 18264 /f
```
